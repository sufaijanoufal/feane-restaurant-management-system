# Feane Restaurant Management System

A full-stack restaurant management system built with Node.js, Express.js, MongoDB, and JavaScript.

## Features

* User Authentication (JWT)
* Role-Based Authorization
* Category Management
* Menu Management
* Multiple Image Uploads
* Booking Management
* Testimonials Management
* About Content Management
* Dynamic Menu Filtering
* Responsive Design

## Tech Stack

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication
* Multer

### Frontend

* JavaScript (ES6 Modules)
* Bootstrap
* jQuery
* Isotope.js
* Owl Carousel

## Installation

```bash
npm install
npm start
```
